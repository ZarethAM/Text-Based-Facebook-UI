
# Text-Based Facebook UI

Facebook is a social media platform that allows users to connect with friends by creating profiles,
sharing posts, photos, and videos, and engaging in various online activities. In this program i utlizised a simple text-based algorithm to code the software which will replicate some features of Facebook in C.
